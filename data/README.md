# 사료 데이터 안내

## 1. 샘플 사료 10건 — `sample_sillok.json`

정동·덕수궁 1896~1907 근대사 코스 시범 데모용 사료 10건입니다. 각 항목 구조:

| 키 | 설명 |
|---|---|
| `id` | 고유 식별자 (예: sillok-001) |
| `title` | 사건명 |
| `date` | 사건일 (YYYY-MM-DD) |
| `source` | 출처 (실록 권수·날짜) |
| `place` | 사건 장소 |
| `place_coords` | [경도, 위도] |
| `original_text` | 원문 인용 (짧은 발췌만) |
| `summary` | 요약 (일반인용) |
| `easy_explanation` | 가족 모드용 쉬운 해설 |
| `tags` | 검색용 태그 |
| `related_persons` | 관련 인물 |
| `source_url` | 한국사데이터베이스 또는 외부 링크 |
| `license` | 데이터 라이선스 |

## 2. 데이터 출처·라이선스

본 샘플 데이터는 다음 출처에 근거합니다:

- **공공데이터포털 — 교육부 국사편찬위원회_조선왕조실록 정보_고순종실록 원문 (15053646)**: 이용허락범위 제한 없음 (원문 데이터)
- **국사편찬위원회 한국사데이터베이스**: 검색·열람용 링크. 번역문·해제 등은 별도 라이선스 적용 (상업적 활용 전 사전 질의 필요)
- **정동제일교회 등 부속 자료**: 참고자료로 표시

> ⚠️ 본 샘플의 `original_text`와 `summary`는 RAG 시연을 위해 작성된 **요약·발췌 형태**이며, 본격 출시 전에는 원문 데이터셋과 1:1 대조 검증이 필요합니다.

## 3. 공공데이터 다운로드 — `../scripts/download_sillok.py`

전체 사료를 공공데이터포털에서 직접 다운로드하려면:

```bash
# data.go.kr API 키 발급 후 .env에 설정
python ../scripts/download_sillok.py --period 1895-1907 --place 정동
```

자세한 내용은 `../scripts/README.md` 참조.

## 4. 데이터 확장 시 주의

- 사료를 추가할 때마다 `easy_explanation` 필드는 사람 검수(역사 자문가 또는 본인) 후 입력
- `original_text`는 짧은 인용(50자 이내) 또는 외부 링크로 처리 (저작권 안전)
- `source_url`은 항상 한국사데이터베이스 또는 공공데이터포털 원문 링크
