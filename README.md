# 사초(史草) AI — Streamlit 시범 데모

> **부문**: 제4회 문화체육관광 인공지능·데이터 활용 공모전 / 문화데이터 활용 — 아이디어 부문
> **목적**: 사관(史官) AI 대화 + 사료 증거 카드 + 사실 확인 배지의 핵심 화면을 시연하는 5분짜리 데모

## 1. 빠른 시작 (3분 안에 실행)

### 필수 준비
- Python 3.10 이상
- [console.anthropic.com](https://console.anthropic.com)에서 발급받은 Claude API 키 (`sk-ant-...`)

### 설치 및 실행 (Windows / macOS / Linux 공통)

```bash
# 1) 프로젝트 폴더로 이동
cd "사초AI_streamlit_demo"

# 2) 가상환경 생성·활성화 (선택, 권장)
python -m venv .venv
# Windows
.venv\Scripts\activate
# macOS/Linux
source .venv/bin/activate

# 3) 의존성 설치
pip install -r requirements.txt

# 4) 환경변수 설정
#    .env.example 을 .env 로 복사한 뒤 ANTHROPIC_API_KEY 채워넣기
copy .env.example .env       # Windows
cp .env.example .env         # macOS/Linux
# .env 파일을 메모장/에디터로 열어 sk-ant-... 키 입력

# 5) Streamlit 실행
streamlit run app.py
```

브라우저가 자동으로 열리며 (http://localhost:8501), 사관 AI와 대화할 수 있습니다.

## 2. 데모에서 확인할 수 있는 4가지 핵심 기능

| 기능 | 화면 위치 | 검증 포인트 |
|---|---|---|
| **① 사관(史官) AI 대화** | 메인 채팅 | 시대 어투의 1인칭 대화, 5턴 이상 자연스러운 흐름 |
| **② 사료 증거 카드** | 응답 하단 | 출처·날짜·장소·원문 발췌·원문 링크·라이선스 자동 표시 |
| **③ 사실 확인 배지** | 응답 상단 | `사료 확인됨` (녹색) / `AI 각색` (파랑) / `추정` (주황) 3종 — 배지 라벨은 언어별 자동 번역 |
| **④ 다국어 즉시 변환** | 사이드바 | 한·영·일·중 간체 4개 언어 — UI·에러 메시지·배지 모두 동시 전환 |

### 시장 위치 — 경쟁 포지셔닝 (심사·내부용)

> 이 절은 **심사위원·내부용**입니다. 사용자에게 노출되는 게이트·메인 화면에는 경쟁사명을 적지 않았습니다.

사초 AI는 새로운 시장을 만들지 않습니다. **이미 검증된 두 시장의 결합·로컬라이즈** 모델입니다.
1. **위치기반 야외 방탈출·도시 탐험 게임** (Questo €7.99~/1.5–3h, 퀘스투어 ₩10K·5만 명 검증)
2. **한국사 1차 사료 검증 학습** (현재 시장에 부재)

| 비교 대상 | 그쪽의 강점 | 우리의 차별 (구체적) |
|---|---|---|
| **국내 야외방탈출 / 미션투어**<br>(퀘스투어, 리얼월드, 지자체 축제형) | 시나리오 완성도, ₩10K 가격대 검증, 정동·독립운동 테마 보유 | • AI **무한 출제** (사전 제작 시나리오 한계 극복)<br>• 1차 **사료 검증 + 출처 링크**<br>• 거의 무료 (Haiku 4.5 기준 1문제 ~₩4)<br>• 설치 불필요 PWA·즉시 시작 |
| **무료 오디오가이드**<br>(KTO Odii, 박물관·궁궐 해설, 유튜브) | 무료, 다국어, 공신력 | • **능동적 퀴즈** (수동 청취 → 사초 적립·연승)<br>• 게임이 끝난 뒤에도 다시 하고 싶은 동기<br>• 사료 카드로 한걸음 더 들어가는 학습 |
| **글로벌 위치기반 게임**<br>(Geocaching Adventure Lab, Questo, Actionbound, Goosechase) | 위치별 단서·다국어(29개), 도시 탐험 가격(€7.99~), DIY 빌더 | • **GPS·현장 체크인 불필요** (즉시 플레이)<br>• 한국사 데이터셋 **사전 큐레이션 90건**<br>• **학설 다양성** 명시 (단일 정답 강요 X)<br>• 한국 1차 사료 직접 인용 |

**핵심 결론**: 사초 AI = 검증된 도시 탐험 게임 모델 × 한국사 사료 검증 × AI 무한 출제 × 학설 다양성. 한국형 위치기반 학습 게임 시장의 첫 사료 검증형 솔루션.

### 보조 기능 (심사·시연 어필용)

| 기능 | 화면 위치 | 용도 |
|---|---|---|
| 가족 / 일반 모드 | 사이드바 | 가족 모드는 `easy_explanation` 우선 + 짧은 문장·쉬운 어휘 시스템 프롬프트 적용 |
| 토큰·예상비용 메타 | 응답 하단 | 입력/출력 토큰 + ₩ 환산 비용 (Haiku 4.5 가격 기준) |
| RAG 검색 점수 패널 | 응답 하단 (옵션) | top-k 매칭 점수·매칭 토큰 노출 — 환각 억제 근거 시각화 |
| 사료 위치 미니맵 | 응답 하단 (옵션) | 카드의 `place_coords`를 지도에 핀 표시 |
| 대화 JSON 내보내기 | 사이드바 | 시연 결과 수집·캡처 첨부용 |
| 자동 재시도 | 내부 | RateLimit·일시 네트워크 오류 시 지수 백오프 2회 재시도 |

## 5. 자주 묻는 질문

**Q. Claude API 키 발급 비용은?**
A. 가입 시 무료 크레딧 제공. **Haiku 4.5** 기준 1회 대화당 약 0.1~0.3원 수준이라 5분 데모는 1~2원이면 충분합니다 (Sonnet 4.5 대비 1/3 가격).

**Q. 샘플 사료 외 다른 시기·장소도 가능한가?**
A. `data/sample_sillok.json`에 JSON 항목을 추가하면 즉시 검색됩니다. `scripts/download_sillok.py`로 공공데이터포털에서 대량 다운로드 가능.

**Q. 사료에 없는 내용을 물으면 어떻게 동작하나?**
A. RAG 검색 결과가 비거나 점수가 낮으면 시스템 프롬프트에 따라 "확인 불가" 응답 또는 `추정` 배지가 부여됩니다 (환각 위험 억제 프로토콜).

**Q. 본 데모와 정식 PWA의 차이는?**
A. 본 데모는 사초 AI의 **3가지 핵심 화면(사관 대화 + 사료 증거 카드 + 사실 확인 배지)** 만 시연합니다. 정식 출시 PWA는 추가로 ① 지도·GPS, ② 카메라 Vision 검증, ③ 결제·다국어 검수, ④ 분기·결말, ⑤ 배리어프리, ⑥ 엔딩 카드를 포함합니다. (사업계획서 §2·§8 참조)

## 6. 라이선스·면책

- 본 데모 코드는 사초 AI 응모팀 내부 활용 목적 (공모전 시연용)
- `data/sample_sillok.json`의 사료 데이터는 공공데이터포털 「조선왕조실록 정보」(이용허락범위 제한 없음) 기반
- LLM 응답은 RAG 검증을 거치지만 100% 정확성을 보장하지 않습니다. 학술·정책 결정에 활용하기 전 인간 전문가 검수가 필요합니다.
